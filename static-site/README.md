# Properties 4 Creations - Static Website

A clean, fast static website showcasing Properties 4 Creations' Section 8 qualified rental housing for veterans and their families.

## Features

- ✅ **Veteran-Focused**: Section 8 qualified properties specifically for military families
- ✅ **Professional Design**: Clean, trustworthy interface with military-inspired styling
- ✅ **Lead Generation**: Contact forms and application CTA buttons
- ✅ **Property Showcase**: Featured rental listings with pricing and location info
- ✅ **Mobile Responsive**: Works perfectly on all device sizes
- ✅ **Static Approach**: No complex backend - fast loading and easy deployment

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation
```bash
cd static-site
npm install
```

### Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build
```bash
npm run build
npm run start
```

## Deployment Options

### Vercel (Recommended)
1. Fork or clone this repository
2. Connect to Vercel and deploy from the `static-site` folder
3. Automatic HTTPS and custom domain support

### Netlify
1. Drag the `static-site` folder to Netlify for instant deployment
2. Uses Vite build output for optimal performance

### Traditional Hosting
1. Build the project: `npm run build`
2. Upload the `out` folder to any static hosting service
